Como trabajamos

Cada proyecto se trabaja de forma personalizada para encontrar la mejor opcion segun el espacio, la necesidad y el resultado buscado.

1

Nos contas que necesitas

Puede ser grama bahiana, cercos vivos, riego, nivelacion o una obra completa.

2

Nos pasas fotos, medidas o zona

Con esa informacion se puede orientar mejor la propuesta y definir la mejor opcion para cada caso.

3

Evaluamos la mejor opcion

Se define la solucion segun el espacio, el tipo de trabajo y la terminacion buscada.

4

Coordinamos el trabajo

Una vez resuelta la cotizacion, se avanza con entrega, colocacion o la obra completa.

Preguntas frecuentes

Respuestas a consultas frecuentes sobre servicios, productos y formas de trabajo.

Realizan cotizaciones por WhatsApp?

Si. Las consultas y cotizaciones se realizan por WhatsApp segun el tipo de trabajo, medidas, zona y necesidad puntual.

Trabajan con obra completa?

Si. Se puede consultar por una resolucion integral que incluya grama, cercos vivos, nivelacion, riego y terminaciones.

Puedo consultar por grama bahiana y cercos vivos?

Si. Se pueden pedir cotizaciones por productos puntuales o por una propuesta mas completa para el espacio.

El valor depende de medidas y zona?

Si. El valor puede variar segun medidas, zona, cantidad y tipo de trabajo a realizar.

Se realizan envios?

Si. Realizamos envios en todo Gran Buenos Aires y alrededores, segun disponibilidad y tipo de pedido.

Como se toman los pedidos?

Los pedidos se toman con seña del 50% y el saldo restante se abona al recibir las plantas.

Se puede coordinar segun cada necesidad?

Si. Cada trabajo puede evaluarse segun el lugar, el uso del espacio y el tipo de terminacion buscada.
